# Project 4: Machine Language

## 專案概述

這個專案讓學生接觸底層的機器語言編程，具體內容包括：

### 主要目標
1. **親身體驗低階編程**：使用 Hack 電腦的組合語言編寫程式
2. **熟悉 Hack 指令集**：為下一個專案（建立 Hack 電腦）做準備
3. **了解組譯過程**：為第六個專案（建立組譯器）打基礎

### 使用的工具
1. **編輯器**：用於編寫組合語言程式
2. **組譯器**：將符號化的組合語言轉換為二進制機器碼
3. **CPU 模擬器**：執行生成的機器碼，模擬電腦硬體的行為

## 學習步驟

### 第一步：觀察與執行現有程式
學生需要先執行和分析幾個現有的組合語言程式：
- **Add.asm**：簡單的加法程式
- **Max.asm**：找出兩個數的最大值
- **Rect.asm**：在螢幕上畫矩形
- **Pong.asm**：經典的乒乓球遊戲（約 28,000 行機器碼）

### 第二步：編寫自己的程式
學生需要編寫兩個程式：

1. **Mult.asm**（算術任務）
   - 輸入：R0 和 R1（RAM[0] 和 RAM[1]）
   - 輸出：計算 R0 * R1，結果存到 R2
   - 條件：假設 R0 ≥ 0, R1 ≥ 0, 且乘積 < 32768

2. **Fill.asm**（輸入/輸出任務）
   - 運行無限迴圈監聽鍵盤
   - 當按下任意鍵時，將整個螢幕變黑
   - 當沒有按鍵時，將整個螢幕清空

## 專案意義

這個專案是從 **硬體層面（前三個專案）** 過渡到 **軟體層面** 的關鍵步驟：

- **前三個專案**：建立邏輯閘、ALU、記憶體等硬體元件
- **這個專案**：使用這些硬體支持的指令集進行編程
- **後續專案**：將建立完整的電腦系統、編譯器等
